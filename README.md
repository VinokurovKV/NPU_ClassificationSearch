## Документация проекта

Проект посвящён разработке алгоритмов поиска в таблицах классификации сетевого процессора и моделированию процесса маршрутизации и коммутации пакетов.

В рамках основного задания реализован алгоритм поиска по таблице MAC-VLAN с использованием бинарного дерева поиска. Реализованы операции вставки и поиска, загрузка таблиц классификации, обработка входных пакетов и их распределение по выходным портам с формированием файлов результатов.

В дополнительной части проекта реализован алгоритм поиска по наибольшему совпадению префикса в таблице FIB, а также собственная реализация B-дерева для хранения классификационных таблиц (FIB, ARP, MAC-VLAN). Добавлена расширенная логика классификации пакетов с последовательной обработкой таблиц и обработкой ситуаций отсутствия данных.

Проект включает:
- реализацию бинарного дерева поиска;
- реализацию B-дерева;
- алгоритм классификации пакетов по портам;
- тестирование производительности структур данных;
- автоматическое тестирование с использованием pytest.
